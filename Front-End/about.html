Welcome!



Interactive Student Games is a student-created website for students with disabilities. On this website, there are several games that are intended to help special-ed teachers enrich their students' in-classroom experience. You can find the following games on our website:

    Color Game

    The user is prompted to click on the shape with the correct color. If they select the correct option, 'correct' will be displayed, and 'incorrect' otherwise.

    Round Up Game

    The player will be given a money-related scenario and will be asked how much they should pay. They will then be given three choices, and they must choose the correct one.

    Memory Game

    The student will be shown a single letter/character to memorize in the span of 8 seconds. After the time to memorize is up, students will be quizzed on what letter was shown and will be able to build up their memory skills.

    Rapid Math Game

    The student will be given a set of basic addition or subtraction problem in a rapid-fire sequence in order to improve their speed calculation skills and their familiarity with numbers.

    Shape Game

    The player will be prompted to click on the correct shape. If their choice is correct, 'correct' will be displayed, and 'incorrect' otherwise
